Caso de uso: Incluir alunos na disciplina
Ator: Administrativo
Pré-condição: usuário está logado e é administrativo
------------------------------------------------

Sistema solicita código da disciplina

Usuário informa código da disciplina

Sistema busca disciplina por código

Se disciplina não existe

	Sistema mostra mensagem

	Caso de uso termina

Sistema busca lista de alunos da disciplina

Enquanto usuário desejar incluir

	Sistema solicita cpf do aluno para cadastrar na disciplina

	Usuário informa cpf do aluno

	Sistema busca aluno por cpf

	Se aluno não existe

		Sistema mostra mensagem de que aluno não existe

	Se aluno existe

		Sistema verifica se aluno já está inscrito na disciplina

		Se aluno já está na disciplina

			Sistema mostra mensagem de que aluno já está incrito na disciplina

		Se aluno não está na disciplina

			Sistema o inclui na disciplina

	Sistema salva disciplina e alunos

Caso de uso termina

-------------------------------------------------
