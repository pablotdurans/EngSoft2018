Caso de uso: Cadastrar departamento
Ator: Administrativo
Pré-condição: usuário está logado e é administrativo
------------------------------------------------

Sistema solicita o nome do departamento

Usuário informa o nome do departamento

Sistema busca departamento por nome

Se departamento com esse nome foi encontrado

	Sistema informa que departamento já existe

	Caso de uso termina

Sistema solicita sigla do departamento

Usuário informa sigla do departamento

Sistema gera código do departamento

Sistema cria novo departamento

Sistema salva departamento

Sistema informa que o departamento foi criado com sucesso

Caso de uso termina
