Caso de uso: Excluir tipo de movimentação
Ator: Administrativo
Pré-condição: usuário está logado e é administrativo
------------------------------------------------

Sistema solicita o nome do tipo de movimentação

Usuário informa o nome do tipo de movimentação

Sistema busca tipo de movimentação por nome

Se tipo de movimentação não foi encontrado

	Sistema informa que tipo de movimentação não existe

	Caso de uso termina

Sistema exclui o tipo de movimentação

Sistema informa que o tipo de movimentação foi removido com sucesso

Caso de uso termina

-------------------------------------------------
