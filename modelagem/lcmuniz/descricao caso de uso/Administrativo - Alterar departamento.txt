Caso de uso: Alterar departamento
Ator: Administrativo
Pré-condição: usuário está logado e é administrativo
------------------------------------------------

Sistema solicita a sigla do departamento

Usuário informa a sigla do departamento

Sistema busca departamento por sigla

Se departamento não foi encontrado

	Sistema informa que departamento não existe

	Caso de uso termina

Sistema solicita dados do departamento (sigla e nome)

Usuário informa dados do departamento

Sistema altera informações do departamento
        
Sistema salva departamento

Sistema informa que o departamento foi alterado com sucesso

Caso de uso termina
